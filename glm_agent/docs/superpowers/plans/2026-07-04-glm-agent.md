# glm-agent Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a standalone, general-purpose local chat CLI (`glm-agent`) that talks to Ollama, as a fallback for when ib-tutor's RAG pipeline or its configured model isn't usable.

**Architecture:** Single Typer command (`glm-agent chat`) running a REPL: read a line, append to an in-memory message history, send the whole history to Ollama's `chat` API (non-streaming — one request/response per turn), print the reply, repeat until `exit`/`quit`/EOF/Ctrl+C.

**Tech Stack:** Python 3.11+, `typer`, `rich`, `ollama` (same stack as `ib-tutor`, proven to work in this environment). `pytest` for tests. `hatchling` build backend, `src/` layout — mirrors `ib-tutor`'s project structure exactly.

## Global Constraints

- No RAG, no citations, no IB grounding rules — this is a plain chat fallback, not a replacement for ib-tutor.
- Default model: `glm4:9b` (5.5GB — already validated as a reasonably-sized local pull). Overridable via `--model`/`-m` flag or `GLM_AGENT_MODEL` env var.
- Non-streaming: call `client.chat(...)` once per turn and print the full reply. No token streaming — keeps the client interface and tests trivial.
- Ollama unreachable or model not pulled → catch the exception, print one actionable line (`run \`ollama serve\` and \`ollama pull <model>\``), and let the user retry the next turn. Never let this crash the REPL.
- This project lives at `local_llm_agent/glm_agent/`, sibling to `ibdp_tutor` and `ibstudy`, as its own git repo with its own `pyproject.toml` and `CLAUDE.md` (matches how `ibdp_tutor` is structured relative to the `local_llm_agent/` parent).

from __future__ import annotations

from typing import Any, Protocol, cast

DEFAULT_MODEL = "glm4:9b"
SYSTEM_PROMPT = "You are a helpful local assistant running fully offline via Ollama."


class ChatClient(Protocol):
    def chat(self, model: str, messages: list[dict[str, str]]) -> dict[str, Any]: ...


class OllamaUnavailable(Exception):
    pass


def send_message(client: ChatClient, model: str, history: list[dict[str, str]]) -> str:
    try:
        response = client.chat(model=model, messages=history)
    except Exception as e:
        raise OllamaUnavailable(
            f"Can't reach Ollama or model '{model}' isn't pulled — "
            f"run `ollama serve` and `ollama pull {model}`."
        ) from e
    return str(response["message"]["content"])

```python
"""Standalone local chat agent over Ollama — fallback for ib-tutor."""

from __future__ import annotations

from typing import Any, Callable, Protocol, cast

import ollama
import typer
from rich.console import Console

DEFAULT_MODEL = "glm4:9b"
SYSTEM_PROMPT = "You are a helpful local assistant running fully offline via Ollama."

app = typer.Typer(add_completion=False)
console = Console()


class ChatClient(Protocol):
    def chat(self, model: str, messages: list[dict[str, str]]) -> dict[str, Any]: ...


class OllamaUnavailable(Exception):
    pass


def send_message(client: ChatClient, model: str, history: list[dict[str, str]]) -> str:
    try:
        response = client.chat(model=model, messages=history)
    except Exception as e:
        raise OllamaUnavailable(
            f"Can't reach Ollama or model '{model}' isn't pulled — "
            f"run `ollama serve` and `ollama pull {model}`."
        ) from e
    return str(response["message"]["content"])


def run_repl(
    model: str,
    client: ChatClient | None = None,
    input_fn: Callable[[], str] | None = None,
) -> None:
    resolved_client: ChatClient = client or cast(ChatClient, ollama.Client())
    resolved_input: Callable[[], str] = input_fn or (lambda: console.input("[bold green]you>[/] "))
    history: list[dict[str, str]] = [{"role": "system", "content": SYSTEM_PROMPT}]

    console.print(
        f"[bold cyan]glm-agent[/] — local chat via Ollama ({model}). Type 'exit' to quit."
    )
    while True:
        try:
            user_input = resolved_input()
        except (EOFError, KeyboardInterrupt):
            console.print()
            return

        stripped = user_input.strip()
        if stripped.lower() in {"exit", "quit"}:
            return
        if not stripped:
            continue

        history.append({"role": "user", "content": stripped})
        try:
            reply = send_message(resolved_client, model, history)
        except OllamaUnavailable as e:
            console.print(f"[bold red]error:[/] {e}")
            history.pop()
            continue

        history.append({"role": "assistant", "content": reply})
        console.print(f"[bold magenta]glm>[/] {reply}")


@app.command()
def chat(
    model: str = typer.Option(DEFAULT_MODEL, "--model", "-m", envvar="GLM_AGENT_MODEL"),
) -> None:
    """Start an interactive local chat session via Ollama."""
    run_repl(model)


if __name__ == "__main__":
    app()
```
